## 3. 핵심 발견

### 3.1 시장별 비교

| 지표 | KOSPI 50 | KOSDAQ 50 | 비교 |

| RSI2 기본 평균수익 | +0.25% | +0.46% | **KOSDAQ이 거의 2배** |
| RSI2 최적화 승률 | 82.8% | 80.0% | 비슷 |
| RSI2 최적화 평균수익 | +1.78% | **+2.84%** | **KOSDAQ이 60% 높음** |
| VWAP 평균수익 | -0.12% | **+0.51%** | **KOSDAQ에서만 유효** |

### 3.2 KOSDAQ에서 RSI2가 더 강한 이유

- KOSDAQ 종목은 변동성이 더 크므로 **과매도 반등 폭이 큼**
- RSI 극단값 도달 시 평균회귀 강도가 KOSPI 대비 강력
- 단, 손절 시 손실도 커질 수 있으므로 ATR 기반 손절 필수

### 3.3 양 시장 공통 최적 파라미터

**RSI(5) < 10, SMA(100), 청산 SMA(3)**이 KOSPI/KOSDAQ 모두에서 1위

```
[매수 조건]
  RSI(5) < 10              → 5일 RSI가 극단적 과매도
  종가 > SMA(100)          → 중기 상승추세 유지 중

[매도 조건]
  종가 > SMA(3)            → 초단기 반등 확인 시 즉시 청산

[손절]
  ATR(14) × 2.5 하방 이탈
```

## 4. 실전 적용 가이드

### 4.1 KOSPI 매매 조건

```
매수: RSI(5) < 10 AND 종가 > SMA(100)
매도: 종가 > SMA(3)
손절: ATR(14) × 2.5 하방
기대: 승률 ~83%, 평균수익 ~+1.8%
```

### 4.2 KOSDAQ 매매 조건

```
매수: RSI(5) < 10 AND 종가 > SMA(100)
매도: 종가 > SMA(3)
손절: ATR(14) × 2.5 하방
기대: 승률 ~80%, 평균수익 ~+2.8%
```

### 4.3 KOSDAQ VWAP 돌파 (보조 전략)

```
매수: 전일 종가 < VWAP → 당일 종가 > VWAP AND 거래량 > 20일 평균 × 1.5
매도: 3거래일 후 종가
손절: VWAP 하방 재이탈
기대: 승률 ~35%, 평균수익 ~+0.5%, PF 1.28
```

### 4.4 리스크 관리

- 포지션당 최대 리스크: 계좌의 1%
- 동시 보유: 최대 4~6종목
- SMA(200) 하방 시장에서는 신규 진입 자제
- 거래비용 고려: KOSPI/KOSDAQ 약 0.4% (수수료 + 세금)

## 5. 주의사항

⚠️ 과거 성과가 미래 수익을 보장하지 않습니다.

- 최적화 파라미터는 3~6개월마다 재검증 필요
- Walk-Forward 검증에서 기본 파라미터 열화 ~20%p 확인됨
- 실거래 전 최소 1개월 페이퍼 트레이딩 권장
- 소액 시작 후 점진적 확대
- 생존자 편향(survivorship bias) 포함 가능
